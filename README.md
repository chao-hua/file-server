# fileserver

## 安装

```
npm i -g fileserver
```

## 使用

```
fileserver # 当前目录为服务器根目录

fileserver -p 8888 # 设置端口号为 8888

fileserver -h localhost # 设置 host 为 localhost

fileserver -r /user # 设置根目录为 /user
```
