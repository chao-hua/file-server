# jasmine-server

## 安装

```
npm i -g jasmine-server
```

## 使用

```
jasmine-server # 当前目录为服务器根目录

jasmine-server -p 8888 # 设置端口号为 8888

jasmine-server -h localhost # 设置 host 为 localhost

jasmine-server -r /user # 设置根目录为 /user
```
